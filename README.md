# 💰 Virtual Currency Library System

🔗 **Live Demo:** https://vcls-yash-malhotra.vercel.app/

Virtual Currency Library System (VCL) is a React-based cryptocurrency dashboard that displays **live market data, trending coins, and historical price charts**.

---

## 🚀 Features

- 📊 Live cryptocurrency data (price, rank, market cap, 24h change)
- 🌍 Global currency switcher (INR, USD, EUR)
- 🔍 Searchable and paginated coins table
- 🔥 Trending coins carousel
- 📈 Historical price charts with multiple time ranges
- 📄 Detailed coin view with market insights (ATH/ATL, description, etc.)

---

## 🧠 Key Concepts Used

- API Integration (CoinGecko)
- React Context (global state management)
- Dynamic routing
- Data visualization (charts)
- Component-based architecture

---

## ⚙️ How It Works

- Fetches cryptocurrency data from the CoinGecko API
- Displays real-time and historical data dynamically
- Allows users to explore individual coins in detail
- Updates UI based on selected currency and timeframe
